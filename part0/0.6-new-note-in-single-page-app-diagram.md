# Exercise 0.6: New note in Single page app

```mermaid
sequenceDiagram
    participant browser
    participant server

    Note right of browser: User writes a note and clicks Save

    Note right of browser: JavaScript calls e.preventDefault() to block submit default
    Note right of browser: JavaScript creates a note object
    Note right of browser: JavaScript updates the DOM and displays the new note

    browser->>server: POST /exampleapp/new_note_spa
    Note right of browser: Content-Type: application/json
    Note right of browser: JSON body contains content and date

    server-->>browser: 201 Created

    Note right of browser: Browser stays on the same page
    Note right of browser: No redirect and no page reload
```
