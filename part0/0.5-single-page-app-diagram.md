# Exercise 0.5: Single page app diagram

```mermaid
sequenceDiagram
    participant browser
    participant server

    Note right of browser: User opens the SPA page

    browser->>server: GET /exampleapp/spa
    server-->>browser: 200 OK, HTML document

    browser->>server: GET /exampleapp/main.css
    server-->>browser: 200 OK, CSS file

    browser->>server: GET /exampleapp/spa.js
    server-->>browser: 200 OK, JavaScript file

    Note right of browser: Browser executes spa.js

    browser->>server: GET /exampleapp/data.json
    server-->>browser: 200 OK, JSON containing the notes

    Note right of browser: JavaScript updates the DOM and displays the notes
```
